# 🌈 417_KeyGuide v0.9.0-beta

🌍 **Multilingual UI Supported**

このアプリは、キーボードとマウスの入力ログを画面上に表示するツールです。<br>
さらに、登録したショートカット一覧を表示させることもできます。

PCソフトの使い方を覚えたい方、チュートリアル動画を投稿したい方にオススメです！

## ✨ 機能紹介

- **入力ログ表示**
  キーボードやマウスのクリック、スクロールを画面に表示。ショートカットの説明もつけられます。

- **チートシート**
  自作したショートカットキーの一覧を、ウィンドウ表示、または全画面表示。AIにリストを作らせれば簡単！

- **マウス透過**
  マウスが近づくと、キーのログ表示が透過し、作業の邪魔になりません。

- **高いカスタマイズ性**
  デザインや機能について、細かく設定可能です。

- **多言語対応**
  日本語、英語、他にも全世界の言語に翻訳可能です。

<br>

## 🚀 使い方

1. [リリースページ](../../releases) から最新版をダウンロードします。
2. [Wiki](../../wiki) の「使い方」を参考に、アプリを起動します。

> ⚠️ **注意事項**  
> キーボードやマウスの入力検知ツールという性質上、ウイルス対策ソフトに誤検知される場合があります。  
> 本アプリは**完全ローカル動作**であり、外部への通信は一切行いませんのでご安心ください。
